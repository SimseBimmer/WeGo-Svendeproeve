# Kom i gang

## Installation

```bash
npm install react-router-dom
cp .env.example .env
```

## Frontend

```bash
cd frontend
npm i
npm run dev
```

## Backend

```bash
cd backend
npm i
npm run dev
cp .env.example .env
npx prisma migrate dev
npm run seed
```
